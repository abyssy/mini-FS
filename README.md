# mini-FS
a mini file system

打开exe文件后在命令行执行

用户指令：

create	创建空间

mount	挂载空间

fmt	格式化当前空间

dr	显示文件目录：dir

cp	复制文件：copy

dl	彻底删除文件

del	删除文件（之后可以恢复）

map	显示指定文件在空间所占用的全部块号

opt	优化空间，单个文件连续存放

att	显示文件属性

help	帮助信息

rewrite	编辑文件

rst	恢复回收站中的文件

empty_bin	清空回收站

add	新建文件

rename	重命名文件

tp	显示文件内容

close	退出 mini-FS 返回操作系统
